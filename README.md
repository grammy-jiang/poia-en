# Psychology of Intelligence Analysis — structured HTML edition

Richards J. Heuer, Jr., *Psychology of Intelligence Analysis*, CIA Center for the Study of
Intelligence, 1999. As a work of the United States federal government, the original text is in
the public domain (17 U.S.C. § 105).

This edition was extracted from the original PDF and rebuilt as a continuous, sectioned,
source-anchored HTML book with PageTether, a local-first PDF structuring pipeline.
The source PDF ships in `assets/original.pdf`; `manifest.json` records the content hashes of
every file in this immutable release.

Live at <https://grammy.jiang.is/poia-en/>. The Chinese translation of the same book is at
<https://grammy.jiang.is/poia-zh/>.
